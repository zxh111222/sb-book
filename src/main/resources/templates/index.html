ISBN 公开信息查询接口

ISBN 简介

国际标准书号(international Standard Book Number)，简称ISBN，是专门为识别图书等文献而设计的国际编号。ISO于1972年颁布了ISBN国际标准，并在西柏林普鲁士图书馆设立了实施该标准的管理机构--国际ISBN中心。采用ISBN编码系统的出版物有:图书、小册子、缩微出版物、盲文印刷品等。

接口详细

基本说明:

接口地址: http://localhost:8080/book/isbn/{ISBN}?apikey={apiKey}

返回格式: json

请求方式: get

请求示例: http://localhost:8080/book/isbn/9787115249494?apikey=1234567890

请求参数说明:

名称 类型 必填 说明
ISBN 字符串 必填 ISBN号
apikey 字符串 必填 apikey
